# Medical Store Website Specification

## Project Overview

Build a modern, responsive website for a local medical store.

The purpose is to:

- Build trust with customers
- Showcase available services
- Display medicines and healthcare products
- Show contact information
- Allow customers to easily contact the store
- Mobile-first responsive design

This is NOT an e-commerce website.
